# Usage
```
cwl-runner fasta_diff.cwl
```

# Equivalent command: 
```
fasta_diff example_file/old.fa example_file/new.fa -o match.tsv -r report.txt
```
(e.g.)
```
fasta_diff Bombus_terrestris.1.0.fa GCF_000214255.1_Bter_1.0_genomic.fna -o match.tsv -r report.txt
```
